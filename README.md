Robustness Testing under Missing and Noisy Data

Author: Primna Toms

University: University of Europe for Applied Sciences (UE)

Project Overview
This project investigates Responsible AI approaches for waste classification using machine learning and computer vision techniques.

Research Questions
RQ1
How does missing, noisy, or imbalanced data affect the fairness of AI models across different demographic or geographic groups?

RQ2
Do more robust AI models require substantially more computational energy, and is the trade-off justified?

RQ3
How stable are AI explanations when data becomes noisy, incomplete, or shifted?

RQ4
Which drift-detection techniques can identify changing environmental conditions before prediction performance significantly degrades?

RQ5
Can uncertainty estimates and confidence warnings improve human decision-making when AI predictions become unreliable?
